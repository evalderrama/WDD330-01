Notes:

  • Variables: We can use the keywords "const" and "let" to declare variables in addition to var. Non-primitive data types can still be mutated.
  • Variables: With the const keyword the variable cannot change its value. With let, another value can be reassigned to the variable.
  • Scope refers to where a constant or variable is accessible by the program.
  • Scope: Values of variables declared with const and let only exists inside the block they are declared in.
  • Global Scope: keep the number of global variables to a minimum. There is a risk of confusion or clash of variables.
  • Template Literals are a special types of string.
  • Template Literals: The JavaScript expression is placed inside the curly braces with a $ character in front of them.
    const name = `Siri`;
    `Hello ${ name }!`;
    Arrays:
    const avengers = ['Captain America', 'Iron Man', 'Thor', 'Hulk', 'Hawkeye', 'Black Widow'];
  • The pop() method removes the last item from an array.
    avengers.pop();
  • The shift() method removes the first item in the array.
    avengers.shift();
  • The push() method appends a new value to the end of the array.
    avengers.push('Thor');
  • The unshift() method appends a new item to the beginning of the array.
    avengers.unshift('Captain America');
  • The concat() method can be used to merge an array with one or more arrays:
    avengers.concat(['Hulk','Hawkeye', 'Black Widow']);
Finding if a Value is in an Array
  • indexOf() method return the index of the first occurrence
    avengers.indexOf('Iron Man');
  • includes() method returns a boolean value depending on whether the array contains a particular element or
    avengers.includes('Iron Man');
