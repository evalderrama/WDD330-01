Modern JavaScript Development
  • jQuery library: can reduce several lines and get the same result using a single line of code.
  • Advantages of Libraries: used and tested by lots of people, there are lots of online documentation, reduce code writing time.
  • Disadvantages of Libraries: you must include the library code as well as your own code, which increases the amount of code a website must download. Speed differences can be barely noticeable, although there are occasions when using a library is a poor choice for some operations.
  • Libraries that have emerged as the most popular so far are React and Vue.js.
