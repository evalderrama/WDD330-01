Notes:

Objects
  • We can define methods for an object. We can define a method as an assignment or inside the object.

    As an "assignment":
    user.sayHi = function() {
        alert("Hello!");
    };


    Inside the method:
    Normal form Short form
    user = {
        sayHi: function() {
           alert("Hello");
        }
    };     		user = {
        sayHi() { // same as "sayHi: function(){...}"
           alert("Hello");
        }
    };
  • “this” in methods: To access the object, a method can use the this keyword.
  • Arrow functions have no “this”. For instance, Arrow functions uses “this“ from the outer method.
  • An object can have values of type strings, numbers, booleans, arrays and functions.
  • Great way to create an object:

    const name = 'Iron Man';
    const realName = 'Tony Stark';

    // short ES6 way
    const ironMan = { name, realName };
  • The parse() method takes a string of data in JSON format and returns a JavaScript object:
    JSON.parse(batman);
  • The stringify() method does the opposite, taking a JavaScript object and returning a string of JSON data, as can be seen in the example:
    JSON.stringify(wonderWoman);
  • Object can be nested.
  • The Regex object can be used to create regular expressions.
DOM
  • Document.links returns a node list of all the < a > elements
  • The insertBefore() method will place a new element before another element
  • It's possible to update CSS with DOM
  • CSS property names that are separated by dashes must be written in camelCase notation
Events
  • The target property returns a reference to the node that fired the event.
